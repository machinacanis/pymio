#  pyMio

_基于 [PIL](https://github.com/python-pillow/Pillow) 和 [opencv-python](https://github.com/opencv/opencv-python) 封装的图像处理工具 / 模板引擎_

## 这是什么？

pyMio 是一个图像处理工具，也是一个模板引擎，将繁琐的图像处理操作大幅封装简化，并且提供了强大的模板格式化功能，绘制图片so easy！
